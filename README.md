# Simple SQLite ORM - 'sORM'

Allows to use Python objects instead of SQLite statements.

## Installation

You can install 'sORM' with these commands:
```bash
    $ mkdir <installation dir>
    $ cd /<installation dir>
    $ git clone git://github.com/max-kim/sorm.git
    $ cd sorm
```

## Requirements

- Python 3.6 (or over)
